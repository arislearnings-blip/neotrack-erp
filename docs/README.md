# NeoTrack ERP

NeoTrack ERP is a Manufacturing, Inventory, Sales, Purchasing and Accounting ERP system.

## Project Objectives

* Manufacturing Management
* Inventory Management
* Purchasing Management
* Sales Management
* Financial Accounting
* Production Planning
* Reporting & Analytics

## Core Modules

1. Dashboard
2. User Management
3. Chart of Accounts
4. Customers
5. Suppliers
6. Inventory
7. Warehouses
8. Purchasing
9. Sales
10. Manufacturing
11. General Journal
12. Ledgers
13. Trial Balance
14. Profit & Loss
15. Balance Sheet
16. Reports
17. Settings

## Planned Features

* Auto Numbering
* Role Based Security
* Document Relationships
* BOM Management
* Production Orders
* Inventory Tracking
* Financial Statements
* PDF Printing
* Excel Export
* Search & Filters
* Audit Trail
